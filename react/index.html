About The Course

We’ll show you how you can leverage React in a modern development environment whilst building up an administrative ecommerce application that will help visually demonstrate best practices and provide you with a scaffold for your own work.

This course takes root in real-world use cases and challenges and includes coverage on more than just the core React framework. We show you how to unit test complex enterprise-grade ReactJS applications faster, how you can to render screens, how you can fetch data from the server, as well as e-mail authoring, and building a quick iOS version of our page. It has a bit of everything for the experienced developer who wants a broad overview of what is possible with React.

By the end you’ll have built an administrative ecommerce application fully equipped with a dashboard, a list of orders, a digest e-mail, and an iOS version.

Save time and write more expressive JavaScript in ES6, whilst joining a growing number of developers who are leveraging the flexibility of React.
